API Documentation

To access the blog data, you will need to use your API Key. Below is the guide on how to use it:

Getting All Blogs

Endpoint: GET /hgcBackendBlogs/users/outer/api/blogs/user/{{ "\{" }}username{{ "\}" }}

Headers:

  • API-Key: Your API key

Example Request:

    GET /hgcBackendBlogs/users/outer/api/blogs/user/johndoe
    Headers: "{{ "\{" }}"API-Key": "your-api-key-here"{{ "\}" }}"

Response

If the API Key is valid, you will receive a list of blogs associated with the user:

    [
      {{ "\{" }}"id": "blogId1", "title": "Blog Title 1", "author": "johndoe", "content": "Blog content here...", "date": "2024-06-07T00:00:00.000Z"{{ "\}" }},
      {{ "\{" }}"id": "blogId2", "title": "Blog Title 2", "author": "johndoe", "content": "Blog content here...", "date": "2024-06-07T00:00:00.000Z"{{ "\}" }}
    ]

If the API Key is invalid, you will receive a 401 Unauthorized status code.
